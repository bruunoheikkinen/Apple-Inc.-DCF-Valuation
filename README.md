# Apple Inc. — 3-Statement Model & DCF Valuation

An integrated 3-statement financial model and dynamic DCF valuation for Apple Inc. (AAPL), built to assess the intrinsic value of Apple Inc. and evaluate whether the stock is fairly valued relative to the current market price.

## Overview

This model links the income statement, balance sheet, and cash flow statement into a single dynamic framework, driven by a dedicated Assumptions sheet. The DCF derives intrinsic value using a CAPM-based WACC and a Gordon Growth terminal value, cross-checked against a trading comparables (EV/EBITDA) approach.

## Key Highlights

- **3-Statement Model** — fully linked income statement, balance sheet, and cash flow statement, driven by a single Assumptions sheet
- **DCF Valuation** — CAPM-based WACC, unlevered free cash flow projection, Gordon Growth Model terminal value
- **Comparable Companies Analysis** — EV/EBITDA benchmarking against relevant peers to sanity-check the DCF-implied exit multiple
- **Sensitivity Analysis** — two-way data table varying WACC and terminal growth rate
- **Executive Summary** — football field valuation chart summarizing DCF, comps, and bull/bear scenarios
